🔍 วิธีหา Windows SID

1. กดปุ่ม Windows + R บนคีย์บอร์ดเพื่อเปิดหน้าต่าง Run

2. พิมพ์คำว่า cmd แล้วกด Enter

3. คัดลอกคำสั่งด้านล่างนี้ ไปวางในหน้าต่างสีดำแล้วกด Enter:

wmic useraccount get name,sid

4. คัดลอกรหัสที่อยู่บรรทัดเดียวกับชื่อ User ของคุณ (ขึ้นต้นด้วย S-1-5-...) มาใส่ในช่องสมัคร
